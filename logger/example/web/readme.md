# logger/example/web

Import path: `github.com/global-torque/go-common/logger/example/v2/web`

Executable Echo web example for the logger package.

## Use For

- Seeing request-context logging in an Echo handler.
- Checking error stack output in HTTP code.

## Do Not Use For

- Library imports in services. This is a `main` package.

## Configuration

Uses normal logger env such as `LOG_LEVEL` and `LOG_CONSOLE`.
